Константы
===========

Модуль:

    sphere.logs.constants

Набор статусов для логгирования

```
    LOGS_LEVEL = (
        ('INFO', 'INFO'),
        ('DEBUG', 'DEBUG'),
        ('WARNING', 'WARNING'),
        ('ERROR', 'ERROR'),
    )

    LOGS_STATUS = (
        ('active', 'Активный'),
        ('resolved', 'Решена'),
        ('ignore', 'Игнорируема'),
    )
```
